# IS2 - Kata 5
Quinta *kata* de Ingeniería del Software 2 del Grado Ingeniería Informática (ULPGC)


### Ejercicio
> Partiendo de la kata anterior, se deberá reimplementar la aplicación para incorporar persistencia con SQLite y carga de datos desde una fuente remota.
En esta nueva versión, el programa deberá implementar dos operaciones principales:
> 1. Carga de datos desde la fuente externas a la base de datos SQLite.
> 2. Generación del histograma a partir de los datos almacenados en la base de datos.


### Código
Creación de una base de datos a partir del dataset de siempre.

Lectura de la base de datos creada.

Uso considerable del principio _tell don't ask_.

Se puede argumentar que la base de datos sigue el patrón un _singleton_.

### Créditos
El dataset en cuestión: [github.com/lgreski/pokemonData](https://github.com/lgreski/pokemonData/blob/master/Pokemon.csv)

* [SQLite v3.51.0.0](https://mvnrepository.com/artifact/org.xerial/sqlite-jdbc/3.51.0.0)
